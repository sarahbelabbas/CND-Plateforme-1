# Documentation technique RX et Gamma

Cette documentation est disponible en version interactive:

- Ouvrir `index.html` pour visualiser les abaques numerises RX/Gamma.
- Survol + clic pour figer un point et lire le facteur Q.
- Lecture du facteur K (film AGFA) dans le panneau de controle.
